# Carpeta de imágenes de técnicas - ArtBook

Coloca aquí las imágenes que se usarán para ilustrar cada técnica.

## Nombres de archivos esperados

Para que el frontend las encuentre automáticamente, usa **exactamente** estos nombres:

- `lapiz-grafito.jpg` → Técnica **Lápiz Grafito**
- `acuarela-basica.jpg` → Técnica **Acuarela**
- `oleo-clasico.jpg` → Técnica **Óleo**
- `carboncillo-artistico.jpg` → Técnica **Carboncillo**

Puedes usar JPG o PNG, pero si cambias la extensión, también debes actualizar el campo `image_url` en el backend (`backend/src/models/techniquesModel.js`).

## Recomendaciones

- Resolución sugerida: entre **800x600** y **1200x800**
- Peso recomendado: menos de **500 KB** por imagen
- Imágenes horizontales se ven mejor en las tarjetas
