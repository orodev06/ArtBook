# ArtBook Frontend

Interfaz web para visualizar técnicas de dibujo.

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Características

- Diseño responsivo con Tailwind CSS
- Paleta morada (#7C3AED) y blanca
- Lista de técnicas con tarjetas
- Indicador de dificultad (Básico, Intermedio, Avanzado)
- Información de materiales
